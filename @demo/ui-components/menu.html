MENU

.menu -menu-nav

Menus are vertical list of links or buttons for actions and navigation.

Add a container element with the menu class. And add child elements with the menu-item class. You can separate menu items with a divider. Menus also have Form controls (checkbox, radio and checkbox) support.

Add -menu-nav to use with .bar or other nav container.
