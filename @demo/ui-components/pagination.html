PAGINATION

.pagination -page-item -page-link -page-prev -page-next -pagination-dots -link-box -page-item-subtitle -page-item-title -active -disabled

Add a container element with the 

pagination
class. And add child elements with the 
-page-item
class. The 
-page-item
with the 
-active
class will be highlighted. You can add the disabled class to the page-item to have unclickable page links.
