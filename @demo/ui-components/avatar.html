AVATAR

.avatar .avatar-{$size} .avatar-icon .avatar-presence -online -busy -away

Add the avatar class to element. There are 4 additional sizes available, including:

  • .avatar-xs --avatar-size-xs: 1rem;
  • .avatar-sm --avatar-size-sm: 1.5rem;
  • .avatar-lg --avatar-size-lg: 3rem;
  • .avatar-xl --avatar-size-xl: 4rem;

By default, the avatar size is 2rem. For users who don't have profile pictures, you may use their initials for avatars. Add the avatar class and avatar-{$size} class to an element. Use the data-initial attribute to show the name inside the avatar.

Avatar icon

Avatar presence

Avatars support presence indicators. You can add an "i" element with the avatar-presence class, and add -online, -busy or -away class for different status colors. The default is gray which means offline.

                            /* css classes */
.avatar {
    display: inline-block;
    font-size: var(--avatar-size) / 2;
    height: var(--avatar-size);
    width: var(--avatar-size);
    background: var(--avatar-bg-color-rgb);
    border-radius: var(--avatar-radius);
    color: var(--avatar-color);
    font-weight: var(--avatar-font-weight);
    line-height: var(--avatar-line-height);
    margin: 0;
    position: relative;
    vertical-align: middle;
}
.avatar.avatar-xs {
    font-size: var(--avatar-size-xs) / 2;
    height: var(--avatar-size-xs);
    width: var(--avatar-size-xs);
}
.avatar.avatar-sm {
    font-size: var(--avatar-size-sm) / 2;
    height: var(--avatar-size-sm);
    width: var(--avatar-size-sm);
}
.avatar.avatar-lg {
    font-size: var(--avatar-size-lg) / 2;
    height: var(--avatar-size-lg);
    width: var(--avatar-size-lg);
}
.avatar.avatar-xl {
    font-size: var(--avatar-size-xl) / 2;
    height: var(--avatar-size-xl);
    width: var(--avatar-size-xl);
}
.avatar img {
    position: relative;
    border-radius: var(--avatar-radius);
    height: 100%;
    width: 100%;
    z-index: var(--avatar-index);
}
.avatar .avatar-icon, .avatar .avatar-presence {
    background: var(--avatar-presence-bg-color-rgb);
    bottom: 14.64%;
    height: 50%;
    padding: var(--avatar-padding);
    position: absolute;
    right: 14.64%;
    transform: translate(50%, 50%);
    width: 50%;
    z-index: var(--avatar-index);
}
.avatar .avatar-presence {
    background: var(--avatar-presence-bg);
    box-shadow: var(--avatar-presence-shadow);
    border-radius: 50%;
    height: 0.5em;
    width: 0.5em;
}
.avatar .avatar-presence.\-online {
    background: var(--liquid-color-success);
}
.avatar .avatar-presence.\-busy {
    background: var(--liquid-color-danger);
}
.avatar .avatar-presence.\-away {
    background: var(--liquid-color-warning);
}
.avatar[data-initial]::before {
    color: currentColor;
    content: attr(data-initial);
    left: 50%;
    position: absolute;
    top: 50%;
    transform: translate(-50%, -50%);
    z-index: var(--avatar-index);
}
