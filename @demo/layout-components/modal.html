MODAL

PREVIEW

.modal -modal-container -modal-header -modal-body -modal-footer -modal-overlay -active -modal-sm -modal-lg -modal-fullwidth -modal-fullheight tooltip

Add a container element with the modal class. And add a real container modal-container and overlay modal-overlay inside it. You can add child elements with the modal-header, -modal-body and -modal-footer - any or all of them. 

-anim-2 -anim-3 -anim-4 -anim-5

If you want a diferent animation loading, add -anim-2 -anim-3 -anim-4 -anim-5 to class .modal. Animation can be configurated with theme variables (see theme variables).
