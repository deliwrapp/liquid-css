Tables

Tables include default styles for tables and data sets.

Add the table class to any table element. The class will add padding, border and emphasized table header.

Default tables

Striped tables

Use the table-striped class to table to add zebra striped style. For hoverable table rows, you can add the -hover class to enable hover style.

Use the active class to make element highlighted.
