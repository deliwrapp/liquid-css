TILES

PREVIEW

.tile -tile-content -tile-action -tile-icon -tile-title -tile-subtitle

Tiles are repeatable or embeddable information blocks.

Add a container with the tile class. And add child elements with the -tile-icon, -tile-content or/and -tile-action classes. The -tile-icon and -tile-action are optional. There are -tile-title and -tile-subtitle classes for title and subtitle text styles.

COMPACT TILES

There is compact version of Tiles component, which is often used as contact and file info blocks. Add the -tile-compact class to the container tile. The tile-icon, tile-content and tile-action will be vertically centered.
